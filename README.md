# MIPS Calculator on MARS

## Project Information

* Created On: 8 May 2026
* Project Type: Assembly Language Project
* Platform: MARS (MIPS Simulator)

---

## Overview

This project is a basic calculator implemented using MIPS assembly language and executed on the MARS simulator.

The program allows users to perform fundamental arithmetic operations such as addition, subtraction, multiplication, and division through console input/output.

It demonstrates low-level programming concepts including registers, system calls, branching, and control flow.

---

## Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division of two numbers
* User input handling
* Console-based output

---

## Concepts Covered

* MIPS instruction set
* Register usage
* Arithmetic operations
* Branching and looping
* System calls (syscall)
* Input/Output handling

---

## How to Run

1. Open MARS simulator

2. Load the assembly file:
   File -> Open -> select the .asm file

3. Assemble the code:
   Click "Assemble"

4. Run the program:
   Click "Run"

5. Follow the on-screen instructions
